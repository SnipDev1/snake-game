## Key bindings:
1) W - Move Up
2) A - Move Left
3) S - Move Down
4) D - Move Right
5) Esc - Quit

1) Download Python from official site
2) Upgrade pip
```
pip install --upgrade pip
```
3) Install "keyboard"
```
pip install keyboard
```
4) Start main.py from console - python main.py
